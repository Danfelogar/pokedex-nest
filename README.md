# Execution en developer mode

1. Clone the repository
2. Execute for install all dependencies
```
pnpm install
```
3. have Nest CLI installed
```
npm i -g @nestjs/cli
```
4. raise up data base
```
docker-compose up -d
```

5. remember update .gitignore for ignore mongo file

## Used Stacks
* Mongo DB
* Nest
